# To-Do App en Python

Aplicación de consola para gestionar tareas con persistencia en JSON.

## Funcionalidades

- Agregar tareas
- Ver tareas
- Eliminar tareas
- Marcar tareas como completadas

## Tecnologías

- Python
- JSON

## Estructura del proyecto

app/
- main.py
- tareas.py
- ui.py

data/
- tareas.json (ignorado)

## Cómo ejecutar

```bash
python app/main.py
